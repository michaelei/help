消息类：com.eg.intf3.msg.RemoteControl

远程操作设备，有多种操作可以执行。参考下面的示例代码

示例代码-重启门锁

//指定主控器唯一标识符和设备通讯码
String ctrlUUID = "14:06:30:00:04:AB";
//如果要重启主控器，只需将deviceCode=0
int deviceCode = 100;
// 创建远程控制消息对象
RemoteControl msg = new RemoteControl(ctrlUUID, deviceCode);
// 设置操作为：重启门锁
msg.setControlAction(RemoteControl.ControlAction_Device_Reset);
//下发消息并设置监听器
console.dispatchTask(msg, new OneTaskStateChangeAdapter(){
@Override
public void onSuccess(Task task) {

}

@Override
public void onFail(Task task) {

}
});
