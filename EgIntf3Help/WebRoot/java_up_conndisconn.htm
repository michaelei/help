public void onConnectionOpened(String remoteIP, int port, String ctrlUUID)

该方法会在主控器建立Tcp连接并告知EgIntf3时触发。

remoteIP：主控器的IP地址。

port：主控器的本次Tcp连接端口号。

ctrlUUID：主控器唯一标识符。如果为null，则此时仅仅表示有Tcp连接上来，但是并不能确定连接来自于哪个主控器。

示例代码

public void onConnectionOpened(String remoteIP, int port, String ctrlUUID) {

if(ctrlUUID==null || ctrlUUID.equals(""))
    return;
System.out.println("主控器 "+ctrlUUID+" 联机。执行后续处理动作......");
}

public void onConnectionClosed(String remoteIP, int port, String ctrlUUID) {

if(ctrlUUID==null || ctrlUUID.equals(""))
    return;
System.out.println("主控器 "+ctrlUUID+" 断线。执行后续处理动作......");
}
