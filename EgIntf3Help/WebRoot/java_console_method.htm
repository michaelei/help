public long dispatchTask(BasicMsg msg,OneTaskStateChangeListner listner)

下发控制任务命令到主控器

msg: 各种下行消息

listner: 针对该消息的任务监听器，即任务成功或失败后的处理，需自定义处理内容，参考“下行控制任务机制说明”

public boolean isCtrlOnline(String ctrlUUID)

判断某个主控器是否在线

ctrlUUID: 主控器唯一标识符

public boolean isStarted()

判断控制台是否已经启动

public void setInitTaskID(long initTaskID)

设置任务的初始编号。每次控制台重启，任务的初始编号都恢复为257开始，如果需要任务编号不重复，则可以在控制器启动之前，通过本方法设置起始编号。

initTaskID: 任务的起始编号，默认257，该编号用于通讯过程中的去重。

public void setTaskTimeout(long timeout)

设置任务超时时间，调用该方法后，此后任务的超时都为新设置的时间。

timeout: 单位毫秒，默认60秒，即 60000。

public void setMaxPopedomCountPerPacket(int maxCount)

设置每一包下载开门权限所能容纳的最大权限条数，默认30条，一般不改动
