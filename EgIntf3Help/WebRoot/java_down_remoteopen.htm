消息类：com.eg.intf3.msg.RemoteOpenLock

远程打开门锁

示例代码

//指定主控器唯一标识符和设备通讯码
String ctrlUUID = "14:06:30:00:04:AB";
int deviceCode = 100;
//构建远程开门控制对象
RemoteOpenLock msg = new RemoteOpenLock(ctrlUUID, deviceCode);
//下发消息并设置监听器
console.dispatchTask(msg, new OneTaskStateChangeAdapter(){
@Override
public void onSuccess(Task task) {

}

@Override
public void onFail(Task task) {

}
});
