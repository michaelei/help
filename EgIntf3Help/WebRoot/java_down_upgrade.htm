消息类：com.eg.intf3.msg.UpgradeNotify

通知主控器或门锁等设备进行固件程序升级，当设备收到该升级通知后，会主动上行来获取升级文件内容。

示例代码

//指定主控器唯一标识符和设备通讯码
String ctrlUUID = "14:06:30:00:04:AB";
//如果要升级主控器，则deviceCode=0
int deviceCode = 100;
// 指定升级文件
File file = new File("D:\\TmpDocs\\lock_0109_2100.bin");
// 创建升级通知消息对象
UpgradeNotify msg = new UpgradeNotify(ctrlUUID, deviceCode, file);
//下发消息并设置监听器
console.dispatchTask(msg, new OneTaskStateChangeAdapter(){
@Override
public void onSuccess(Task task) {

}

@Override
public void onFail(Task task) {

}
});
