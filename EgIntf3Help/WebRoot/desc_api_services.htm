API服务的启动

双击RunAsServer/runAsServer.bat启动服务，如下图：

1、填写相应的配置数据：

主控器（Controller）的TCP端口：默认2013，提供给底层主控器连接的使用。
上层应用（APP）的TCP端口：默认2014，提供给上层应用程序使用。
APP连接的用户名和密码，默认都是app，该密码即是APP认证中使用的密码。
软件右侧提供内置的短信猫程序。（后续会有说明）
注意：为了保持了Tcp的连接，APP可以间隔一段时间（5分钟或10分钟）发送任意非XML格式的字符（必须是ASC码字符）给服务器作为心跳包

2、点击“启动”按钮启动服务

3、点击“停止”按钮停止服务
