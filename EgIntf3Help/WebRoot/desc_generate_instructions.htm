写卡数据的辅助生成

当设备处于离线状态或者第一次配置时候，需要将一些数据写入到卡片中，从而实现特定的操作功能，本部分描述了写卡数据内容的生成。（具体操作详见EgIntf3DemoCode demoWriteCard方法）。具体的写卡操作，针对不同的写卡设备，由APP自定义完成，此处仅仅是辅助得到需要写入卡片的字节信息。

每次写卡，都需要设置一个系统密钥：systemKey，该密钥是一个字符串，由APP维护，通过主控器参数设置中的matchCode参数设置到主控器，同时通过密钥设置卡来设置门锁，两者必须一致，即matchCode= systemKey，该卡才可以对相应的门点进行操作。

各种卡类型说明：

卡类型

说明

Peopedom Card

权限卡
用于开门

Systerm Key Set Card

系统密码设置卡
系统中新加入一个门锁，需要通过该卡将系统密钥设置到门锁中

Region Set Card

区域设置卡
设置门锁的所属区域，栋、层、区

Total Control Card

总控卡

Emergency Card

应急卡

Clean Card

清洁卡
权限卡的一种特例

TimeSetCard

时钟卡
当某个门锁处于脱机状态，可以通过该卡调整其内部时钟的时间

各个卡片数据内容的生成对象均在包：com.eg.intf3.writecard中，详细参考demoWriteCard方法
